Europese Code voor elektronische communicatie ()

EU Richtlijn
De Europese Code voor elektronische communicatie richtlijn stelt een set bijgewerkte regels vast voor de regulering van elektronischecommunicatienetwerken (telecommunicatie), telecommunicatiediensten en bijbehorende voorzieningen en diensten.
